Path Walker Game

How to Play

Navigate through the maze to collect letters in the correct order.

Use the arrow keys to move the green ball.

Collect letters in the order: A → C → B

Reach the orange X to complete the level.

Visual Guide

  • Green ball - You
  • Red letters - Uncollected letters
  • Green letters - Collected letters
  • Orange X - End point

Objective

Find the correct path to collect all letters in order and reach the end point.
